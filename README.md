# Nassau Candy Factory-to-Customer Shipping Route Efficiency Analysis

## Project Overview
Transform raw order/shipment data into actionable **route-level logistics intelligence** for Nassau Candy Distributor. Identify efficient vs delayed factory-to-customer routes, geographic bottlenecks, ship-mode performance.

## Key Deliverables
1. **Streamlit Dashboard**: Interactive route analysis + filters + maps + leaderboards
2. **Research Paper**: EDA, insights, recommendations  
3. **Executive Summary**: Stakeholder-ready findings
